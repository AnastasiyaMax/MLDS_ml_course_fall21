## Формула оценки

Итог = Округление(0.5 * ДЗ + 0.3 * ПР + 0.2 * К), \
где ДЗ — средняя оценка за все домашние задания, ПР — средняя оценка за все проверочные работы, К - оценка за курс на Coursera \
Округление арифметическое.
